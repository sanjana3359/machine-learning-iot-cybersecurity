Machine Learning project to predict network flow duration in IoT environments using RT-IoT2022 dataset. 

## Project Overview
This project applies machine learning techniques to predict network flow duration in IoT environments using the RT-IoT2022 dataset. The aim is to analyse network traffic patterns and support cybersecurity analytics.

## Objectives
- Perform data preprocessing and feature scaling  
- Apply multiple regression models  
- Compare model performance using evaluation metrics  

## Models Implemented
- Linear Regression  
- Decision Tree Regressor  
- Random Forest Regressor  
- K-Nearest Neighbors (KNN)  
- Gradient Boosting Regressor  

## Evaluation Metrics
- Mean Absolute Error (MAE)  
- Root Mean Square Error (RMSE)  
- R² Score  

## Results Summary
Linear Regression achieved the highest accuracy (R² ≈ 0.99998), while Gradient Boosting provided a more robust and reliable performance for real-world scenarios.

## Key Insights
- Ensemble models performed better in capturing nonlinear relationships  
- KNN showed lower performance due to high-dimensional data  
- Extremely high accuracy may indicate feature correlation or data leakage  

## Dataset
RT-IoT2022 Dataset  
(Source: https://archive.ics.uci.edu/dataset/942/rt-iot2022 )
